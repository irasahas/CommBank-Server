# CommBank-Server

A RESTful Banking API built using ASP.NET Core Web API and MongoDB Atlas.

## Features
- User Management (CRUD)
- Account Management (CRUD)
- Goal Management (CRUD)
- Transaction Management (CRUD)
- Tag Management (CRUD)
- MongoDB Atlas Integration
- Swagger API Documentation

## Technologies Used
- ASP.NET Core Web API
- C#
- MongoDB Atlas
- Swagger (OpenAPI)
- Git & GitHub

## API Endpoints

### User
- GET /api/User
- GET /api/User/{id}
- POST /api/User
- PUT /api/User/{id}
- DELETE /api/User/{id}

### Account
- GET /api/Account
- POST /api/Account
- PUT /api/Account/{id}
- DELETE /api/Account/{id}

### Goal
- GET /api/Goal
- POST /api/Goal
- PUT /api/Goal/{id}
- DELETE /api/Goal/{id}

### Transaction
- GET /api/Transaction
- POST /api/Transaction
- PUT /api/Transaction/{id}
- DELETE /api/Transaction/{id}

### Tag
- GET /api/Tag
- POST /api/Tag
- PUT /api/Tag/{id}
- DELETE /api/Tag/{id}

## Database
MongoDB Atlas

All endpoints were tested successfully using Swagger UI.
